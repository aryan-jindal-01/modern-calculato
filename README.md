# 🧮 Modern Calculator

A modern and responsive calculator built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Responsive Design
- Modern UI
- Keyboard Support
- AC (All Clear)
- DEL (Delete Last Character)
- Percentage (%)
- Basic Arithmetic Operations
- Error Handling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
modern-calculator/
│── index.html
│── style.css
│── script.js
│── README.md
```
